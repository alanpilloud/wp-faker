Wp-Faker

is a tool that allows to generate random dummy content for WordPress and Acf.

View on GitHub Download .zip Download .tar.gz

About

WP Faker generates quickly a lot of content with random data. These data are generated using the Faker library.

The most interresting thing about it is that WP Faker is able to populate Acf fields.

Usage

Now you need to code...just a little bit

To let the magic happen, you have to build a configuration file.

Just rename or copy config-sample.php and start modifying it to suit your needs.

Config files must respect this pattern : config-[name].php

You can choose which post type to populate, set a title, content, featured images. If you stop it here, you already have a good foundation to generate blog posts.

The most interresting part is the generation of contents for Acf fields. Each value is assign to the Acf field name as defined in your Acf groups.

...and get ready for the magic !

You're ready ! Browse to your /wp-faker directory and generate, generate, generate... Delightful !

Warning

WP Faker usage doesn't suit a production environment and is still in beta version.
